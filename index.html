What I Use

I'm a problem solver who builds full-stack web applications. I use C#, PostgreSQL, ASP.NET MVC, JavaScript, Bootstrap, HTML, and CSS. Here are some highlights:

I use C# on a daily basis to build websites and web applications. You can see examples of my work implementing C# in my projects section.

I use proven design patterns and frameworks to build websites. ASP.NET Core Web and MVC is exactly what I need for most projects. Check out my projects to see more examples

JavaScript is the language of the web. I use vanilla JS and I am comfortable with many JS frameworks such as JQuery. Check out my challenges section for some JS projects.

I use HTML, CSS, and Bootstrap to build beautiful and responsive websites. Bootstrap is my go-to CSS framework. Many shy away from the front-end, but I embrace it.
