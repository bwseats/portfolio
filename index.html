What I Use

I'm a problem solver who builds full-stack web applications. I use C#, PostgreSQL, ASP.NET, JavaScript, Bootstrap, HTML, and CSS. Here are some highlights:

I use C# on a daily basis to build websites and web applications. You can see examples of my work implementing C# in my projects section.

I use proven design patterns and frameworks to build websites. ASP.NET Core Web and MVC is exactly what I need for most projects. Check out my projects to see more examples

JavaScript is the language of the web. I use vanilla JS and I am confortable with many JS frameworks such as JQuery. Check out my challenges section for some JS projects.

I use HTML, CSS, and Bootstrap to build beautiful and responsive websites. Bootstrap is my go-to CSS framework. Many shy away from the front-end, but I embrace it.

I use git and github for all my projects. Git is the most used source control framework on the planet. I also employ CI/CD for publishing.

SQL is the language of the database. My projects all utilize a database. I have used Postgres on all of them, and I can write SQL on any DB platform.
